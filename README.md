Vimayla (Work in Progress)

Vimayla is a work-in-progress web application built with Next.js, TypeScript, Tailwind CSS, Clerk, Supabase, and OpenAI.

The platform is designed to generate flashcards, quizzes, and study materials automatically using AI based on the content submitted by users through a form. Currently, the project is under active development.

Features (Planned)

AI-powered flashcard generation

AI-powered quizzes and tests

User data storage using Supabase

User authentication and management with Clerk

Responsive and modern UI with Tailwind CSS

Integration with OpenAI for AI content generation

⚠️ Note: Vimayla is currently a Work in Progress. Some features may not be fully functional yet.

Tech Stack

Next.js – Frontend framework

TypeScript – Type-safe development

Tailwind CSS – Styling and layout

Clerk – Authentication and user management

Supabase – Backend, database, and storage

OpenAI – AI content generation
